## ¿Qué es Microsoft 365 Copilot?

**Microsoft 365 Copilot** es un asistente de productividad impulsado por IA que se integra perfectamente con las aplicaciones de Microsoft 365. Combina el poder de los grandes modelos de lenguaje (LLMs) con los datos de tu organización para ayudarte a trabajar de manera más eficiente y creativa.

Con Microsoft 365 Copilot, puedes:
- Generar contenido en Word, Excel, PowerPoint y Outlook.
- Analizar datos y crear información automáticamente.
- Resumir reuniones y conversaciones en Teams.
- Automatizar tareas repetitivas en las aplicaciones.
- Obtener ayuda contextual basada en la información de tu organización.

Microsoft 365 Copilot se integra estrechamente con Microsoft Graph y Azure Active Directory, convirtiéndolo en un componente clave de la estrategia de productividad moderna de Microsoft.

<Callout title="Punto clave">
Microsoft 365 Copilot es un asistente de IA integrado en las aplicaciones de Microsoft 365, diseñado para mejorar la productividad a través de la automatización inteligente y la generación de contenido.
</Callout>

## ¿Qué es Copilot Studio?

**Copilot Studio** es una plataforma de código bajo que te permite crear agentes de IA personalizados y extender las capacidades de Microsoft 365 Copilot para tus necesidades empresariales específicas.

Específicamente:
- Puedes crear **agentes conversacionales personalizados** que comprendan tus procesos empresariales.
- Los agentes pueden configurarse para acceder a **fuentes de datos específicas** dentro de tu organización.
- La plataforma proporciona **plantillas predefinidas** e **interfaz de arrastrar y soltar** para un desarrollo fácil.
- Los agentes personalizados pueden desplegarse en aplicaciones de Microsoft 365 e integrarse con Power Platform.

Copilot Studio transforma la forma en que las organizaciones pueden personalizar experiencias de IA mientras mantienen seguridad y cumplimiento de nivel empresarial.

<Callout title="Punto clave">
Copilot Studio permite a las organizaciones crear agentes de IA personalizados que extienden Microsoft 365 Copilot con conocimiento y capacidades específicas del negocio.
</Callout>

---

## ¿Cuál es la diferencia entre Microsoft 365 Copilot y Copilot Studio?

**Microsoft 365 Copilot** es el asistente de IA listo para usar que viene integrado con las aplicaciones de Microsoft 365, proporcionando características generales de productividad.

**Copilot Studio**, por otro lado, es una plataforma de desarrollo que te permite crear agentes de IA personalizados adaptados a tus necesidades empresariales específicas.

| Solución              | Microsoft 365 Copilot | Copilot Studio        |
|-----------------------|------------------------|----------------------|
| Propósito             | Productividad general  | Desarrollo de IA personalizado |
| Usuarios objetivo     | Usuarios finales      | Profesionales de TI/desarrolladores |
| Nivel de personalización | Limitado            | Altamente personalizable |
| Fuentes de datos      | Microsoft Graph       | Conexiones personalizadas |
| Despliegue            | Preintegrado          | Despliegue personalizado |

Ambas soluciones trabajan juntas para proporcionar un ecosistema de IA integral donde las características generales de productividad se mejoran con capacidades personalizadas y específicas del negocio.

<Callout title="Punto clave">
Microsoft 365 Copilot proporciona características de IA listas para usar, mientras que Copilot Studio te permite construir agentes de IA personalizados. Se complementan para una solución de IA completa.
</Callout>

## Licenciamiento de Microsoft 365 Copilot Pro

**Microsoft 365 Copilot Pro** es una licencia premium que proporciona capacidades de IA mejoradas y características adicionales más allá de la oferta estándar de Microsoft 365 Copilot.

### Requisitos de licenciamiento

Para usar Microsoft 365 Copilot Pro, necesitas:
- **Licencia base de Microsoft 365** (E3, E5, Business Premium o equivalente)
- **Licencia complementaria de Microsoft 365 Copilot Pro** (costo adicional por usuario por mes)
- **Azure Active Directory** para autenticación y gestión de usuarios
- Acceso a la **API de Microsoft Graph** para integración de datos organizacionales

### Características clave de Copilot Pro

**Capacidades de IA mejoradas:**
- Acceso prioritario a GPT-4 Turbo durante horas pico
- Tiempos de respuesta más rápidos y rendimiento mejorado
- Capacidades avanzadas de razonamiento y comprensión

**Soporte extendido de aplicaciones:**
- Integración completa en todas las aplicaciones de Microsoft 365
- Características mejoradas en Word, Excel, PowerPoint, Outlook y Teams
- Acceso a Copilot en servicios adicionales de Microsoft

**Seguridad y cumplimiento avanzados:**
- Protección de datos de nivel empresarial
- Registro de auditoría e informes de cumplimiento
- Controles de residencia de datos
- Integración de protección avanzada contra amenazas

### Consideraciones de despliegue

**Configuración administrativa:**
- Requiere privilegios de administrador global para la configuración inicial
- Gestión centralizada de licencias a través del centro de administración de Microsoft 365
- Configuración de políticas para controles de acceso y uso de datos

**Gestión de usuarios:**
- Asignación de licencias por usuario individual
- Controles de acceso basados en roles
- Capacidades de monitoreo y reportes de uso

**Gobernanza de datos:**
- Se respetan los límites de datos organizacionales
- Ningún dato se usa para entrenar los modelos de IA subyacentes
- Cumplimiento con regulaciones de la industria (GDPR, HIPAA, etc.)

### Caso de uso

Imagina que recibes un correo electrónico complejo del departamento de finanzas solicitando justificaciones de presupuesto para tus proyectos de TI. En lugar de redactar manualmente una respuesta, puedes usar Microsoft 365 Copilot Pro en Outlook para analizar el contenido del correo electrónico, comprender los requisitos financieros específicos mencionados y generar una respuesta profesional que incluya datos de presupuesto relevantes de tus correos electrónicos anteriores y documentos de SharePoint.

### Estructura de costos

| Componente | Descripción | Facturación |
|-----------|-------------|---------|
| Licencia base | Microsoft 365 E3/E5 o Business Premium | Mensual/Anual |
| Complemento Copilot Pro | Capacidades adicionales de IA | Por usuario/mes |
| Almacenamiento | Almacenamiento adicional para procesamiento de IA | Según sea necesario |

<Callout title="Punto clave">
Microsoft 365 Copilot Pro requiere tanto una licencia base de Microsoft 365 como una licencia complementaria adicional de Copilot Pro, proporcionando capacidades de IA mejoradas con seguridad y cumplimiento de nivel empresarial.
</Callout>
