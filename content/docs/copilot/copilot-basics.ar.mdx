## ما هو Microsoft 365 Copilot؟

**Microsoft 365 Copilot** هو مساعد إنتاجية مدعوم بالذكاء الاصطناعي مدمج بسلاسة داخل تطبيقات Microsoft 365. يجمع بين قوة نماذج اللغة الكبيرة (LLMs) وبيانات مؤسستك لمساعدتك على العمل بكفاءة وإبداع.

مع Microsoft 365 Copilot يمكنك:
- توليد محتوى في Word و Excel و PowerPoint و Outlook.
- تحليل البيانات وإنشاء رؤى تلقائياً.
- تلخيص الاجتماعات والمحادثات في Teams.
- أتمتة المهام المتكررة عبر التطبيقات.
- الحصول على مساعدة سياقية تعتمد على معلومات مؤسستك.

يتكامل Microsoft 365 Copilot بعمق مع Microsoft Graph و Azure Active Directory ما يجعله جزءاً رئيسياً في إستراتيجية الإنتاجية الحديثة لدى Microsoft.

<Callout title="خلاصة أساسية">
Microsoft 365 Copilot هو مساعد ذكاء اصطناعي مدمج في تطبيقات Microsoft 365 لتعزيز الإنتاجية عبر الأتمتة الذكية وتوليد المحتوى.
</Callout>

---

## ما هو Copilot Studio؟

**Copilot Studio** منصة منخفضة الشفرة (Low-Code) تسمح لك بإنشاء وكلاء ذكاء اصطناعي مخصصين وتوسيع قدرات Microsoft 365 Copilot وفق احتياجات أعمالك.

تحديداً:
- يمكنك إنشاء **وكلاء محادثة مخصصين** يفهمون عمليات عملك.
- يمكن ضبط الوكلاء للوصول إلى **مصادر بيانات محددة** داخل المؤسسة.
- توفر المنصة **قوالب جاهزة** و **واجهة سحب وإفلات** لتسهيل التطوير.
- يمكن نشر الوكلاء المخصصين عبر تطبيقات Microsoft 365 والدمج مع Power Platform.

يُحوِّل Copilot Studio طريقة تخصيص تجارب الذكاء الاصطناعي لدى المؤسسات مع الحفاظ على أمان وامتثال على مستوى المؤسسة.

<Callout title="خلاصة أساسية">
Copilot Studio يمكّن المؤسسات من إنشاء وكلاء ذكاء اصطناعي مخصصين يوسّعون Copilot بذكاء خاص بالأعمال.
</Callout>

---

## ما الفرق بين Microsoft 365 Copilot و Copilot Studio؟

**Microsoft 365 Copilot** هو المساعد الجاهز (Out-of-the-box) المدمج في تطبيقات Microsoft 365 ويقدم ميزات إنتاجية عامة.

**Copilot Studio** من ناحية أخرى منصة تطوير تتيح بناء وكلاء ذكاء اصطناعي مخصصين حسب احتياجات الأعمال.

| الحل | Microsoft 365 Copilot | Copilot Studio |
|------|-----------------------|----------------|
| الهدف | إنتاجية عامة | تطوير ذكاء مخصص |
| المستخدمون المستهدفون | المستخدمون النهائيون | مسؤولو / مطورو تقنية المعلومات |
| مستوى التخصيص | محدود | عالي التخصيص |
| مصادر البيانات | Microsoft Graph | اتصالات مخصصة |
| النشر | مدمج مسبقاً | نشر مخصص |

يعمل الحلّان معاً لتقديم منظومة ذكاء شاملة حيث تُعزَّز الميزات العامة بقدرات مخصصة خاصة بالأعمال.

<Callout title="خلاصة أساسية">
Microsoft 365 Copilot يوفر ميزات ذكاء جاهزة، بينما Copilot Studio يسمح ببناء وكلاء مخصصين؛ يكملان بعضهما لتوفير حل ذكاء متكامل.
</Callout>

## ترخيص Microsoft 365 Copilot Pro

**Microsoft 365 Copilot Pro** ترخيص متميز يقدم قدرات ذكاء موسعة وميزات إضافية تتجاوز العرض القياسي.

### متطلبات الترخيص

لاستخدام Microsoft 365 Copilot Pro تحتاج إلى:
- **ترخيص أساسي Microsoft 365** (مثل E3 أو E5 أو Business Premium أو ما يعادلها)
- **إضافة ترخيص Copilot Pro** (تكلفة إضافية لكل مستخدم شهرياً)
- **Azure Active Directory** للمصادقة وإدارة المستخدمين
- **وصول Microsoft Graph API** لدمج بيانات المؤسسة

### الميزات الرئيسية لـ Copilot Pro

**قدرات ذكاء معززة:**
- أولوية وصول إلى GPT-4 Turbo في أوقات الذروة
- أزمنة استجابة أسرع وأداء محسّن
- قدرات استدلال وفهم متقدمة

**دعم موسع للتطبيقات:**
- تكامل كامل عبر تطبيقات Microsoft 365
- ميزات محسّنة في Word و Excel و PowerPoint و Outlook و Teams
- وصول إلى Copilot في خدمات Microsoft إضافية

**أمان وامتثال متقدم:**
- حماية بيانات على مستوى المؤسسة
- سجلات تدقيق وتقارير امتثال
- ضوابط موقع تخزين البيانات (Data Residency)
- تكامل مع الحماية المتقدمة من التهديدات

### اعتبارات النشر

**إعداد إداري:**
- يتطلب صلاحيات مسؤول عام للإعداد الأولي
- إدارة تراخيص مركزية عبر مركز إدارة Microsoft 365
- ضبط سياسات للتحكم في الوصول للبيانات والاستخدام

**إدارة المستخدمين:**
- إسناد الترخيص لكل مستخدم على حدة
- ضوابط وصول قائمة على الأدوار
- إمكانات مراقبة استخدام وتقارير

**حوكمة البيانات:**
- احترام حدود بيانات المؤسسة
- عدم استخدام البيانات لتدريب النماذج الأساسية
- امتثال للوائح الصناعة (GDPR، HIPAA، ...)

### حالة استخدام

تخيل تلقيت رسالة بريد معقدة من قسم المالية تطلب تبرير الميزانية لمشاريع التقنية لديك. بدل إعداد الرد يدوياً يمكنك استخدام Copilot Pro في Outlook لتحليل المحتوى وفهم المتطلبات المالية المذكورة وتوليد رد مهني يتضمن بيانات ميزانية ذات صلة من رسائل سابقة ومستندات SharePoint.

### هيكل التكلفة

| المكوّن | الوصف | الفوترة |
|---------|-------|---------|
| الترخيص الأساسي | Microsoft 365 E3/E5 أو Business Premium | شهري / سنوي |
| إضافة Copilot Pro | قدرات ذكاء إضافية | لكل مستخدم / شهر |
| التخزين | تخزين إضافي لمعالجة الذكاء | حسب الحاجة |

<Callout title="خلاصة أساسية">
Microsoft 365 Copilot Pro يتطلب ترخيص Microsoft 365 أساسي بالإضافة لإضافة Copilot Pro، مقدماً قدرات ذكاء محسّنة مع أمان وامتثال على مستوى المؤسسة.
</Callout>
