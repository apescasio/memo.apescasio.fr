## Inscribir una Segunda VM

Para confirmar que el perfil de Autopilot, el package de Google Chrome y la política de administrador local se aplican correctamente al grupo dinámico, inscribí una segunda VM.

![Segunda Inscripción VM 1](/images/intune/intune-lab/enrollment-second-VM-lab-intune-1.png)

Puedo ver el perfil como Asignado en el dispositivo, así que continué el aprovisionamiento reiniciando la VM e iniciando sesión.

![Segunda Inscripción VM 2](/images/intune/intune-lab/enrollment-second-VM-lab-intune-2.png)

El package de Google Chrome se instaló correctamente, y el registro de transcripción también está disponible en `C:\ProgramData\Lab\Intune`.

![Segunda Inscripción VM 3](/images/intune/intune-lab/enrollment-second-VM-lab-intune-3.png)

Pude iniciar sesión con la segunda cuenta "IT@LABAPES01.onmicrosoft.com" y también pude ejecutar como administrador, ¡así que la política funcionó!

![Segunda Inscripción VM 4](/images/intune/intune-lab/enrollment-second-VM-lab-intune-4.png)
