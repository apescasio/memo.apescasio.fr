## ما هو Intune؟

**Microsoft Intune** حل سحابي لإدارة الأجهزة المحمولة (MDM - Mobile Device Management) وإدارة التطبيقات المحمولة (MAM - Mobile Application Management). يسمح للمؤسسات بإدارة الحواسيب المكتبية، المحمولة، الهواتف، والأجهزة اللوحية عن بُعد سواء كانت شخصية (BYOD) أو مملوكة للشركة.

مع Intune يمكنك:
- نشر التهيئات (Wi-Fi، VPN، القيود، إلخ)
- دفع التطبيقات (Win32، MSI، تطبيقات المتجر)
- تطبيق سياسات الأمان
- إدارة تحديثات ويندوز، تعريف الحلقات، والتحكم بالتنزيل/التثبيت
- ضمان الامتثال عبر قواعد تقييم

يتكامل Intune بعمق مع Azure Active Directory و Microsoft 365 مما يجعله عنصراً محورياً في استراتيجية الإدارة الحديثة.

<Callout title="الخلاصة">
Intune منصة سحابية لإدارة الأجهزة والتطبيقات مثالية للبيئات الحديثة واستراتيجيات BYOD.
</Callout>

## ما هو Windows Autopilot؟

**Windows Autopilot** ميزة مكمّلة لـ Intune تتيح إعداد ونشر أجهزة ويندوز آلياً بدون صورة (Image) تقليدية.

باختصار:
- يجب أولاً تسجيل الجهاز الجديد في خدمة **Windows Autopilot** (عبر سكربت أو من الشركة المصنِّعة OEM)
- يُسند للجهاز ملف تعريف Autopilot في Intune
- الفني يُشغّل وضع **White Glove** بالضغط على زر ويندوز 5 مرات ثم يختار «Provision with Windows Autopilot» لتحضير الجهاز
- يصبح الجهاز جاهزاً للاستخدام بعد خطوات بسيطة

يُبسّط Autopilot تجربة الإعداد ويقلل العبء على فرق الدعم.

<Callout title="الخلاصة">
Autopilot يؤتمت نشر ويندوز لكنه يتطلب تسجيل الجهاز مسبقاً وإسناد ملف تعريف. يمكن استخدامه للنشر الذاتي أو التحضير المسبق قبل التسليم.
</Callout>

## ما الفرق بين Intune و System Center Configuration Manager (SCCM)؟

**System Center Configuration Manager (SCCM/MECM)** حل أقدم داخلي (On-Prem) يُستخدم في البيئات التقليدية لإدارة أجهزة ويندوز (جرد، نشر أنظمة، تصحيح، ...).

**Microsoft Intune** بالمقابل سحابي بالكامل يركّز على المرونة والإدارة الحديثة متعددة المنصات.

| الحل          | SCCM            | Intune                           |
|---------------|-----------------|----------------------------------|
| البنية        | داخلية On-Prem  | سحابة Cloud                     |
| نشر الأنظمة   | قوي             | محدود (عبر Autopilot)           |
| التغطية       | غالباً ويندوز    | متعددة المنصات (iOS/Android/macOS/Windows) |
| سيناريو BYOD  | محدود           | مدعوم بقوة                      |

اليوم يمكن تشغيل الاثنين معاً عبر **Microsoft Endpoint Manager** الذي يوحّد الإدارة في واجهة هجينة.

<Callout title="الخلاصة">
SCCM داخلي، Intune سحابي. يمكن دمجهما للاستفادة من مزايا العالمين.
</Callout>
