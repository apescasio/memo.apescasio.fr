## ¿Qué es `git` / `GitHub`?

**git** es una herramienta poderosa que actúa como una máquina del tiempo para tus proyectos. Mantiene un registro de cada cambio que haces en tu código, permitiéndote revertir a un estado anterior si es necesario. Piénsalo como un libro de registro o un álbum de fotos para tu proyecto, donde cada commit representa un hito importante o una "instantánea" del progreso de tu trabajo.

Además de eso, git es un "director de orquesta" para el trabajo colaborativo. Cuando múltiples personas trabajan en el mismo proyecto, asegura que todos puedan contribuir sin sobrescribir el trabajo de los demás. Gracias a su función de ramificación, te permite explorar diferentes ideas o características sin arriesgar el proyecto principal.

**GitHub**, por otro lado, es una plataforma basada en la cloud que actúa como una biblioteca para tus proyectos de git. Te permite almacenar, compartir y colaborar en código con otros desarrolladores mientras proporciona herramientas para rastrear contribuciones y gestionar proyectos de manera eficiente.
