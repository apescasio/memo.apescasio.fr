## ما هو `git` / `GitHub`؟

**git** أداة قوية تعمل كآلة زمنية لمشاريعك. تُسجِّل كل تغيير تجريه على الشفرة بحيث يمكنك العودة لحالة سابقة عند الحاجة. فكر فيه كدفتر سجلات أو ألبوم صور لمشروعك، كل التزام (commit) يمثل محطة مهمة أو "لقطة" لتقدّم العمل.

إضافةً إلى ذلك git بمثابة "منسّق" للعمل التعاوني. عندما يعمل عدة أشخاص على نفس المشروع يضمن أن يسهم الجميع دون أن يكتب أحدهم فوق عمل الآخر. وبفضل ميزة الفروع (branching) تستطيع تجربة أفكار أو خصائص جديدة بدون المخاطرة بالفرع الرئيسي.

**GitHub** من جهة أخرى منصة سحابية تعمل كـ "مكتبة" لمشاريع git خاصتك. تمكّنك من تخزين الشفرة ومشاركتها والتعاون عليها مع مطورين آخرين، وتوفر أدوات لتتبع المساهمات وإدارة المشاريع بكفاءة.
