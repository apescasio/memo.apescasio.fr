## Qu'est-ce que `git` / `GitHub` ?

**git** est un outil puissant qui agit comme une machine à remonter le temps pour vos projets. Il garde une trace de chaque modification que vous apportez à votre code, vous permettant de revenir à un état précédent si nécessaire. Imaginez-le comme un journal de bord ou un album photo pour votre projet, où chaque commit représente une étape importante ou une "photo" de l'évolution de votre travail.

En plus de cela, git est un "chef d'orchestre" pour le travail collaboratif. Lorsque plusieurs personnes travaillent sur le même projet, il s'assure que tout le monde peut contribuer sans écraser le travail des autres. Grâce à ses branches, il permet d'explorer différentes idées ou fonctionnalités sans risquer de casser le projet principal.

**GitHub**, quant à lui, est une plateforme basée sur le cloud qui agit comme une bibliothèque pour vos projets git. Elle permet de stocker, partager et collaborer sur du code avec d'autres développeurs, tout en offrant des outils pour suivre les contributions et gérer les projets efficacement.
